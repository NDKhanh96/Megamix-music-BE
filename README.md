## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Migrations
```bash
# tạo file migrate vào trong root/db/migrations
$ npm run migration:generate --name=<migrations file name>

# thực thi những file migrations chưa sử dụng
$ yarn migration:run
```
